# Tenerife Apartment 522A — Guest Guide

## Jak nasadit na Vercel

1. Jděte na **https://vercel.com/new**
2. Přihlaste se (GitHub / Google / email — zdarma).
3. Klikněte na **"Deploy"** → **"Deploy without Git"** (úplně dole) NEBO přetáhněte celou tuto složku.
   - Pokud máte Vercel CLI: `cd vercel-deploy && vercel --prod`
4. Nastavte název projektu: **`tenerife-apartment-522a`**
5. Klikněte **Deploy**.

Za ~15 vteřin dostanete odkaz:
**https://tenerife-apartment-522a.vercel.app**

## URL struktura

| URL | Jazyk |
|---|---|
| `/` nebo `/en` | 🇬🇧 Anglicky (výchozí) |
| `/cz` | 🇨🇿 Česky |
| `/de` | 🇩🇪 Německy |

## Jak zapnout statistiky (Vercel Analytics)

Po nasazení:
1. Otevřete projekt na dashboardu Vercelu
2. Záložka **"Analytics"** → **"Enable"** (zdarma pro hobby plán)

Jakmile aktivujete, skript už je v kódu připraven a začne sbírat data automaticky.

## Jak aktualizovat obsah v budoucnu

Nejjednodušší: znovu přetáhněte upravenou složku na stejný projekt na Vercelu. Nebo si propojte s GitHub repo pro automatické aktualizace.
